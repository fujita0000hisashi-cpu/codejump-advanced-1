# CodeJump Advanced 1 - Reproduction

CodeJump 上級編 1つ目のレイアウトを React + Typescript + css で再現するプロジェクトです。

---

## 目的

- 純粋なCSSによるデザイン再現力の向上
- レイアウト・構造・装飾の役割分離
- セマンティックHTMLの構造整理
- 2カラムレイアウトの責任分離設計
※ 初回は純粋なCSSで作成する。完成後はSassへ移行予定。

## 技術スタック
- Vite
- React
- TypeScript
- CSS (Sass移行予定)

## Setup
```bash
npm install
npm run dev
```
